Gourmet Catering — W3.CSS-style site for GitHub Pages

Quick upload steps
1) Create a public repo named your-username.github.io
2) Upload index.html and the images folder to the repo root
3) In Settings → Pages, set Source to Deploy from a branch → main → /(root)
4) Visit https://your-username.github.io after a few minutes
5) Add collaborator rkmc0der in Settings → Collaborators

Customize
- Edit index.html: change brand text, owner name, and email
- Replace images in /images with your own photos (same filenames)
